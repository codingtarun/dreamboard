# 🧠 DreamBoard — Your Idea Playground

Welcome to your personal brainstorming command center. This is where **no idea is too crazy**.

## How to Use

1. **Brain-dump freely** — Just tell me your idea. I'll capture it.
2. **I manage the rest** — I categorize, tag, track status, and organize everything.
3. **Develop over time** — Flesh out ideas, add details, combine concepts, or archive what doesn't spark joy.

## Idea Lifecycle

```
[RAW] → [DEVELOPING] → [COMPLETED]
   ↓         ↓
[ARCHIVE] [INCUBATING]
```

| Status | Meaning |
|--------|---------|
| **Raw** | Fresh brain-dump. Unfiltered, chaotic, pure energy. |
| **Developing** | Being fleshed out with details, research, plans. |
| **Incubating** | Good idea, but not the right time. Let it sit. |
| **Completed** | Done! Implemented, launched, or achieved. |
| **Archive** | Abandoned, merged, or no longer relevant. |

## Quick Commands (just say these)

- *"I have a crazy idea..."* — I'll create it in `raw/`
- *"Let's develop the [idea name] idea"* — Move to `developing/`
- *"That idea is done!"* — Move to `completed/`
- *"Not now, maybe later"* — Move to `incubating/`
- *"Kill that idea"* — Move to `archive/`
- *"Show me all my ideas"* — I'll list everything
- *"What are my best ideas?"* — I'll surface top-rated ones
- *"Search ideas about [topic]"* — Find related concepts
- *"Combine [idea A] and [idea B]"* — Merge concepts into something new

## Idea Format

Every idea gets:
- **Title** — Catchy name
- **Status** — Current lifecycle stage
- **Tags** — Topics, domains, vibes
- **Spark Level** — 🔥 (mild) to 🔥🔥🔥🔥🔥 (nuclear)
- **Created** — Date
- **Description** — The core concept
- **Notes** — Developments, research, random thoughts
- **Related Ideas** — Links to other concepts
